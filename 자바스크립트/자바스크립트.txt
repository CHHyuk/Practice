자바스크립트란?

- HTML : 웹의 내용 / CSS : 웹의 디자인 / 자바스크립트 : 웹의 동작 구현
- 주로 웹 브라우저 / 프레임워크를 사용하면 서버 프로그래밍에도 사용
- 현재 사용하는 컴퓨터나 스마트폰의 웹 브라우저 대부분에 자바스크립트 인터프리터가 내장되어 있음

자바스크립트의 특징

- 객체 기반의 스크립트 언어
- 동적이며 타입을 명시할 필요가 없는 인터프리터 언어
- 객체 지향형 프로그래밍과 함수형 프로그래밍을 모두 표현할 수 있음

C언어의 경우 소스파일 작성 > 컴파일 > 실행파일(.exe)
인터프리터 언어는 컴파일 작업을 거치지 않고 소스 코드를 바로 실행할 수 있는 언어를 의미함
자바스크립트는 웹 브라우저에 포함된 자바스크립트 인터프리터가 소스 코드를 직접 해석하여 바로 실행시킴

자바 : 컴파일 언어 / 타입 검사를 엄격하게 함 / 클래스(Class)기반의 객체 지향 언어
자바스크립트 : 인터프리터 언어 / 타입 명시 X / 프로토타입(Prototype)기반의 객체 지향 언어
